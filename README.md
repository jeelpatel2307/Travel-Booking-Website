# Travel Booking Website

This project is a simple implementation of a Travel Booking Website in Java. It allows users to book trips, view their bookings, and manage their travel plans.

## Features

- **Book a Trip:** Users can book a trip by providing their name, destination, start date, and end date.
- **View Bookings:** Users can view their bookings, including the destination, start date, and end date of each trip.

## How to Run

1. Ensure you have Java installed on your system.
2. Clone this repository.
3. Compile the Java files using the command:

```

javac TravelBookingSystem.java

```
4. Run the compiled Java program using the command:

```

java TravelBookingSystem

```

## Usage

- Upon running the program, users will be presented with a menu where they can choose to book a trip, view bookings, or exit the system.
- To book a trip, users need to enter their name, trip destination, start date, and end date.
- To view bookings, users can select the appropriate option from the menu, and their current bookings will be displayed.
